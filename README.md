# chrome-extention-discussions

## 使い方
1. Chromeの拡張機能にインストールする
2. オプションを開く。
3. 用意しておいた、GithubIDと名前の変換一覧のJSONを入力し、保存する。

### GithubIDと名前の変換一覧の作り方

以下のJSONフォーマットにしてください。

```json
{
   "members":[
      {
         "github_id":"penguin4731",
         "name":"がっしー"
      },
      {
         "github_id":"hoge",
         "name":"ほげ"
      },
   ]
}
```
